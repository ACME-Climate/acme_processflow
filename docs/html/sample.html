A sample config file with three cases. Unlike the example config in the repo, this is a real config file provided as an example. Each case uses a different file transfer method. Note the data_types configuration for the 20180215.DECKv1b_abrupt4xCO2.ne30_oEC.edison case, which points it to look in the cases local_path directory.
