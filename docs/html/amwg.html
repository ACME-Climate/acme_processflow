AMWG Diagnostic¶

The AMWG Diagsnostic is the standard atmospheric diagnostic package.

Configuration¶

[global]
...
set_frequency = <some_length>

[[set_jobs]]
    ncclimo = <some_length>
    amwg = <some_length>

[amwg]
# The location of the amwg code
diag_home = /p/cscratch/acme/amwg/amwg_diag

# The directory to copy output to for hosting
host_directory = amwg

Dependencies¶

For each year set that AMWG is configured to run, it requires regridded climatologies created by ncclimo. This means that you have to make sure that for each year_set ncclimo is also set to run. AMWG will wait for ncclimo to finish generating climatologies before starting its run.
